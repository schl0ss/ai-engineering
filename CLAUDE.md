# Claude Code Notes

Claude Code should treat this repository as a vendor-neutral AI engineering portfolio.

- Use `skills/**/SKILL.md` as canonical skill instructions.
- Do not create Claude-only behavior that diverges from the Markdown skill.
- Project-specific Claude commands, subagents, or hooks may be documented as adapters, but the portable workflow belongs in `SKILL.md`.
- Never publish private project context from `private-projects/`.
